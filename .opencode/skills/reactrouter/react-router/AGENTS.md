# React Router Skill

This skill provides all the knowledge necessary to work with React Router.

## When to Use This Skill

Use this skill when the user needs:

- Configure routing in a React application
- Create dynamic routes with parameters
- Handle navigation programmatically
- Implement data loading with loaders
- Handle form submissions with actions
- Set up error boundaries
- Work with React Router v7 (or v6)

## How to Use

The agent should load this skill when it detects:

- References to React Router or react-router
- Route configuration needs
- Navigation between pages
- URL parameters handling
- SPA (Single Page Application) setup

## Main Content

- **Routing**: Configuration, nested routes, dynamic segments
- **Components**: Link, NavLink, Outlet, Navigate, Form
- **Hooks**: useNavigate, useParams, useSearchParams, useLoaderData
- **Data Loading**: loaders, actions, revalidation
- **Data Routers**: createBrowserRouter, createHashRouter
- **Utilities**: redirect, generatePath, matchPath

## Usage Examples

The agent can guide the user through:

1. Initial project setup with React Router
2. Creating routes with parameters
3. Implementing data loading with loaders
4. Handling form submissions with actions
5. Setting up error boundaries
6. Migrating from v6 to v7
7. Choosing between Framework, Data, and Declarative modes
