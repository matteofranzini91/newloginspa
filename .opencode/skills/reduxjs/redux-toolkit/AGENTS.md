# Redux Toolkit Skill

This skill provides all the knowledge necessary to work with Redux Toolkit.

## When to Use This Skill

Use this skill when the user needs:

- Configure a Redux store
- Create slices and reducers
- Handle asynchronous operations
- Implement data fetching with RTK Query
- Write TypeScript code for Redux
- Migrate from traditional Redux to Redux Toolkit

## How to Use

The agent should load this skill when it detects:

- References to Redux or Redux Toolkit
- Global state configuration needs
- React data fetching patterns
- Problems with traditional createStore

## Main Content

- **configureStore**: Store configuration
- **createSlice**: Slice creation with Immer
- **createAsyncThunk**: Async thunks
- **createEntityAdapter**: Entity management
- **createSelector**: Memoized selectors
- **RTK Query**: Data fetching and caching
- **React-Redux Hooks**: useSelector, useDispatch

## Usage Examples

The agent can guide the user through:

1. Initial project setup with RTK
2. Creating slices following best practices
3. Implementing RTK Query for APIs
4. Proper TypeScript typing
5. Async state handling
6. Selector optimization
